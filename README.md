# Football Project

This is a simple football analytics project demonstrating Python, SQL (SQLite), Excel, and Power BI.

## Files

- `football_analysis.py` → Python script that reads `matches.csv`, creates SQLite database, and exports team goals to Excel.  
- `data/matches.csv` → Sample football match data.  
- `football.db` → SQLite database (optional to include).  
- `team_goals.xlsx` → Excel file generated by Python, used in Power BI.  
- `football_dashboard.pbix` → Power BI dashboard (optional to include).  

## How to Run

1. Make sure Python is installed.  
2. Install required packages:  
pip install pandas openpyxl
3. Run the script:  
python football_analysis.py
4. Open `team_goals.xlsx` in Power BI to create charts.
